## 🖥️ How to Run Locally

```bash
git clone https://github.com/kavyasri-singam/cypress-e2e-ci.git
cd cypress-e2e-ci
npm install
npx cypress open   # to open Cypress GUI
npx cypress run    # to run tests in headless mode
